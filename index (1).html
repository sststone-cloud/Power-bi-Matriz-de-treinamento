Fonte de dados (atualização via Excel)

Você tem duas formas de manter este painel atualizado:

  1. Substituir o arquivo dados.csv no repositório do GitHub — exporte a aba "BASE_POWERBI" da planilha como CSV e faça upload substituindo o arquivo.
  2. Usar um link publicado: no Excel Online (OneDrive/SharePoint), abra a aba BASE_POWERBI → Arquivo → Compartilhar → Publicar na Web → escolha "Planilha" e formato CSV → copie o link. Cole abaixo. A partir daí, sempre que você editar e salvar a planilha, este painel refletirá os dados novos automaticamente, sem precisar mexer no GitHub.
